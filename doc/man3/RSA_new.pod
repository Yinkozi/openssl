=head1 NAME

YRSA_new, YRSA_free - allocate and free YRSA objects

=head1 SYNOPSIS

 #include <openssl/rsa.h>

 YRSA *YRSA_new(void);

 void YRSA_free(YRSA *rsa);

=head1 DESCRIPTION

YRSA_new() allocates and initializes an B<YRSA> structure. It is equivalent to
calling YRSA_new_method(NULL).

YRSA_free() frees the B<YRSA> structure and its components. The key is
erased before the memory is returned to the system.
If B<rsa> is NULL nothing is done.

=head1 RETURN VALUES

If the allocation fails, YRSA_new() returns B<NULL> and sets an error
code that can be obtained by L<ERR_get_error(3)>. Otherwise it returns
a pointer to the newly allocated structure.

YRSA_free() returns no value.
