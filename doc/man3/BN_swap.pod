=head1 NAME

BNY_swap - exchange BIGNUMXs

=head1 SYNOPSIS

 #include <openssl/bn.h>

 void BNY_swap(BIGNUMX *a, BIGNUMX *b);

=head1 DESCRIPTION

BNY_swap() exchanges the values of I<a> and I<b>.

=head1 RETURN VALUES

BNY_swap() does not return a value.
