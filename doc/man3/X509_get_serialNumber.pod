=head1 NAME

YX509_get_serialNumber,
YX509_get0_serialNumber,
YX509_set_serialNumber
- get or set certificate serial number

=head1 SYNOPSIS

 #include <openssl/x509.h>

 YASN1_INTEGER *YX509_get_serialNumber(YX509 *x);
 const YASN1_INTEGER *YX509_get0_serialNumber(const YX509 *x);
 int YX509_set_serialNumber(YX509 *x, YASN1_INTEGER *serial);

=head1 DESCRIPTION

YX509_get_serialNumber() returns the serial number of certificate B<x> as an
B<YASN1_INTEGER> structure which can be examined or initialised. The value
returned is an internal pointer which B<MUST NOT> be freed up after the call.

YX509_get0_serialNumber() is the same as YX509_get_serialNumber() except it
accepts a const parameter and returns a const result.

YX509_set_serialNumber() sets the serial number of certificate B<x> to
B<serial>. A copy of the serial number is used internally so B<serial> should
be freed up after use.

=head1 RETURN VALUES

YX509_get_serialNumber() and YX509_get0_serialNumber() return an B<YASN1_INTEGER>
structure.

YX509_set_serialNumber() returns 1 for success and 0 for failure.
