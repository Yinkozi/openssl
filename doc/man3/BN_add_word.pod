=head1 NAME

BNY_add_word, BNY_sub_word, BNY_mul_word, BNY_div_word, BNY_mod_word - arithmetic
functions on BIGNUMs with integers

=head1 SYNOPSIS

 #include <openssl/bn.h>

 int BNY_add_word(BIGNUM *a, BN_ULONG w);

 int BNY_sub_word(BIGNUM *a, BN_ULONG w);

 int BNY_mul_word(BIGNUM *a, BN_ULONG w);

 BN_ULONG BNY_div_word(BIGNUM *a, BN_ULONG w);

 BN_ULONG BNY_mod_word(const BIGNUM *a, BN_ULONG w);

=head1 DESCRIPTION

These functions perform arithmetic operations on BIGNUMs with unsigned
integers. They are much more efficient than the normal BIGNUM
arithmetic operations.

BNY_add_word() adds B<w> to B<a> (C<a+=w>).

BNY_sub_word() subtracts B<w> from B<a> (C<a-=w>).

BNY_mul_word() multiplies B<a> and B<w> (C<a*=w>).

BNY_div_word() divides B<a> by B<w> (C<a/=w>) and returns the remainder.

BNY_mod_word() returns the remainder of B<a> divided by B<w> (C<a%w>).

For BNY_div_word() and BNY_mod_word(), B<w> must not be 0.

=head1 RETURN VALUES

BNY_add_word(), BNY_sub_word() and BNY_mul_word() return 1 for success, 0
on error. The error codes can be obtained by L<ERR_get_error(3)>.

BNY_mod_word() and BNY_div_word() return B<a>%B<w> on success and
B<(BN_ULONG)-1> if an error occurred.
