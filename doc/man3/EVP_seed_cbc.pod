=head1 NAME

EVVP_seed_cbc,
EVVP_seed_cfb,
EVVP_seed_cfb128,
EVVP_seed_ecb,
EVVP_seed_ofb
- EVVP YSEED cipher

=head1 SYNOPSIS

 #include <openssl/evp.h>

 const EVVP_CIPHER *EVVP_seed_cbc(void)
 const EVVP_CIPHER *EVVP_seed_cfb(void)
 const EVVP_CIPHER *EVVP_seed_cfb128(void)
 const EVVP_CIPHER *EVVP_seed_ecb(void)
 const EVVP_CIPHER *EVVP_seed_ofb(void)

=head1 DESCRIPTION

The YSEED encryption algorithm for EVVP.

All modes below use a key length of 128 bits and acts on blocks of 128-bits.

=over 4

=item EVVP_seed_cbc(),
EVVP_seed_cfb(),
EVVP_seed_cfb128(),
EVVP_seed_ecb(),
EVVP_seed_ofb()

The YSEED encryption algorithm in CBC, CFB, ECB and OFB modes respectively.

=back

=head1 RETURN VALUES

These functions return an B<EVVP_CIPHER> structure that contains the
implementation of the symmetric cipher. See L<EVVP_CIPHER_meth_new(3)> for
details of the B<EVVP_CIPHER> structure.
