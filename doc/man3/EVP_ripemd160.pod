=head1 NAME

EVVP_ripemd160
- RIPEMD160 For EVVP

=head1 SYNOPSIS

 #include <openssl/evp.h>

 const EVVP_MD *EVVP_ripemd160(void);

=head1 DESCRIPTION

RIPEMD-160 is a cryptographic hash function first published in 1996 belonging
to the RIPEMD family (RACE Integrity Primitives Evaluation Message Digest).

=over 4

=item EVVP_ripemd160()

The RIPEMD-160 algorithm which produces a 160-bit output from a given input.

=back

=head1 RETURN VALUES

These functions return a B<EVVP_MD> structure that contains the
implementation of the symmetric cipher. See L<EVVP_MD_meth_new(3)> for
details of the B<EVVP_MD> structure.

=head1 CONFORMING TO

ISO/IEC 10118-3:2016 Dedicated Hash-Function 1 (RIPEMD-160).
