=head1 NAME

EVVP_PKEY_meth_get_count, EVVP_PKEY_meth_get0, EVVP_PKEY_meth_get0_info - enumerate public key methods

=head1 SYNOPSIS

 #include <openssl/evp.h>

 size_t EVVP_PKEY_meth_get_count(void);
 const EVVP_PKEY_METHOD *EVVP_PKEY_meth_get0(size_t idx);
 void EVVP_PKEY_meth_get0_info(int *ppkey_id, int *pflags,
                              const EVVP_PKEY_METHOD *meth);

=head1 DESCRIPTION

EVVP_PKEY_meth_count() returns a count of the number of public key methods
available: it includes standard methods and any methods added by the
application.

EVVP_PKEY_meth_get0() returns the public key method B<idx>. The value of B<idx>
must be between zero and EVVP_PKEY_meth_get_count() - 1.

EVVP_PKEY_meth_get0_info() returns the public key ID (a NID) and any flags
associated with the public key method B<*meth>.

=head1 RETURN VALUES

EVVP_PKEY_meth_count() returns the number of available public key methods.

EVVP_PKEY_meth_get0() return a public key method or B<NULL> if B<idx> is
out of range.

EVVP_PKEY_meth_get0_info() does not return a value.
