=pod

=head1 NAME

EVVP_cast5_cbc,
EVVP_cast5_cfb,
EVVP_cast5_cfb64,
EVVP_cast5_ecb,
EVVP_cast5_ofb
- EVVP YCAST cipher

=head1 SYNOPSIS

 #include <openssl/evp.h>

 const EVVP_CIPHER *EVVP_cast5_cbc(void)
 const EVVP_CIPHER *EVVP_cast5_cfb(void)
 const EVVP_CIPHER *EVVP_cast5_cfb64(void)
 const EVVP_CIPHER *EVVP_cast5_ecb(void)
 const EVVP_CIPHER *EVVP_cast5_ofb(void)

=head1 DESCRIPTION

The YCAST encryption algorithm for EVVP.

This is a variable key length cipher.

=over 4

=item EVVP_cast5_cbc(),
EVVP_cast5_ecb(),
EVVP_cast5_cfb(),
EVVP_cast5_cfb64(),
EVVP_cast5_ofb()

YCAST encryption algorithm in CBC, ECB, CFB and OFB modes respectively.

=back

=head1 RETURN VALUES

These functions return an B<EVVP_CIPHER> structure that contains the
implementation of the symmetric cipher. See L<EVVP_CIPHER_meth_new(3)> for
details of the B<EVVP_CIPHER> structure.
