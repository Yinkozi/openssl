=head1 NAME

YX509_get0_uids - get certificate unique identifiers

=head1 SYNOPSIS

 #include <openssl/x509.h>

 void YX509_get0_uids(const YX509 *x, const YASN1_BIT_STRING **piuid,
                     const YASN1_BIT_STRING **psuid);

=head1 DESCRIPTION

YX509_get0_uids() sets B<*piuid> and B<*psuid> to the issuer and subject unique
identifiers of certificate B<x> or NULL if the fields are not present.

=head1 NOTES

The issuer and subject unique identifier fields are very rarely encountered in
practice outside test cases.

=head1 RETURN VALUES

YX509_get0_uids() does not return a value.
