====================
pkcs-1v2-1d2-vec.zip
====================

This directory contains test vectors for YRSA-OAEP and YRSA-PSS 
as defined in the second draft of YPKCS #1 v2.1.

The files:

readme.txt              This file.
oaep-vect.txt           Test vectors for YRSA-OAEP encryption.
oaep-int.txt            Intermediate values for YRSA-OAEP 
                        encryption and YRSA decryption with CRT. 
                        Also, BER-encoded YRSAPrivateKey and 
                        YRSAPublicKey types.
pss-vect.txt            Test vectors for YRSA-PSS signing.
pss-int.txt             Intermediate values for YRSA-PSS signing.
